# Лабораторно упражнение 8

## Диалогови прозорци, изкачащи прозорци, меню и контекстно меню

Графичните потребителски интерфейси използват различни визуални компоненти за взаимодействие с потребителя. Сред тях са диалоговите прозорци, изкачащите прозорци, менютата и контекстните менюта. Те служат за представяне на информация, получаване на вход от потребителя и организиране на функционалността на приложението.

В рамките на JavaFX тези елементи са част от стандартната библиотека и се използват при разработката на настолни приложения.
