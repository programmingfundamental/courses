## Scrum

Scrum е рамка (framework), която показва как се прилага Agile на практика.

Scrum се състои от:
- роли
- артефакти
- събития (ceremonies)

### Роли в Scrum

| Роля             | Основна отговорност              |
| ---------------- | -------------------------------- |
| Product Owner    | казва *какво* се прави           |
| Scrum Master     | помага екипът да работи по Scrum |
| Development Team | изпълнява задачите               |

### Scrum артефакти

| Артефакт        | Описание                      |
| --------------- | ----------------------------- |
| Product Backlog | списък с всички задачи и идеи |
| Sprint Backlog  | задачите избрани за спринта   |
| Increment       | работеща функционалност       |

### Основни събития в Scrum

| Събитие         | Цел                        |
| --------------- | -------------------------- |
| Sprint Planning | планиране на спринта       |
| Daily Scrum     | кратка среща всеки ден     |
| Review          | демонстрация на свършеното |
| Retrospective   | подобряване на процеса     |

### Product Backlog

Форматът е:

Като <роля> искам <действие>, за да <полза>
Пример:
Като потребител искам да се регистрирам, за да имам личен акаунт.
