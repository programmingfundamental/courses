# Понятие за Клас в Java

В обектно-ориентираното програмиране (ООП) **класът** е абстрактно описание (модел) на група обекти от реалния свят, които споделят общи характеристики. Можем да разглеждаме класа като **шаблон** или „чертеж“, който дефинира структурата (**състояния**) и функционалността (**поведение**) на конкретните обекти (инстанции), създадени чрез него.

Класът дефинира **нов потребителски тип данни**. Той описва какви атрибути ще притежават обектите, без да задава конкретни стойности за тях в самия шаблон. Освен данни, класовете съдържат и функции, наречени **методи**, които определят действията, които обектите могат да извършват.

### Декларация на клас
Декларацията на един клас включва следните задължителни елементи:
1. **Запазена дума `class`** – указва началото на дефиницията.
2. **Идентификатор (Име)** – съгласно конвенциите в Java, имената на класовете започват с **главна буква** (PascalCase).
3. **Тяло на класа** – съдържанието на класа се огражда с фигурни скоби `{ }`.

```java
class Dog {
    // Тяло на класа: атрибути, конструктори и методи
}
```

---

## Свойства (Атрибути)

Атрибутите са променливи, дефинирани в тялото на класа, които съхраняват състоянието на обектите. Те се наричат още **член-променливи** или полета на класа.

```java
class Dog {
    String name;  // Име на кучето
    String breed; // Порода
    int age;      // Възраст
    String color; // Цвят
}
```

## Конструктори

**Конструкторът** е специален метод, който се използва за инициализиране на нови обекти. Той се извиква автоматично в момента на създаване на инстанция чрез ключовата дума `new`.

### Изисквания към конструктора:
* Името му трябва да съвпада точно с името на класа.
* Не притежава тип на връщания резултат (дори `void`).
* Може да бъде претоварен (един клас може да има няколко конструктора с различни параметри).

### Видове конструктори:
* **Конструктор по подразбиране (Default Constructor):** Не приема параметри. Ако в класа не е дефиниран нито един конструктор, Java автоматично добавя празен конструктор по подразбиране.
* **Параметризиран конструктор:** Приема аргументи, за да зададе начални стойности на атрибутите при самото създаване на обекта.

```java
class Dog {
    String name;
    String breed;

    // Параметризиран конструктор
    public Dog(String name, String breed) {
        this.name = name;   // 'this' реферира към текущия обект
        this.breed = breed;
    }
}
```
*Забележка: Ако дефинирате собствен конструктор с параметри, Java спира да генерира автоматично конструктора по подразбиране.*

## Поведение (Методи)

Методите дефинират функционалността на обектите. В Java е добра практика атрибутите да бъдат защитени, а достъпът до тях да се осъществява чрез специални методи:

1. **Getter методи (аксесори):** Служат за извличане (четене) на стойността на даден атрибут.
2. **Setter методи (мутатори):** Служат за промяна (запис) на стойността на даден атрибут, като често включват логика за валидация.

```java
// Пример за Getter и Setter
public String getName() {
    return name;
}

public void setName(String name) {
    this.name = name;
}
```

## Методи на класа `Object`

Всеки клас в Java автоматично наследява класа `Object`. За постигане на правилно поведение на нашите обекти, често се налага да пренапишем (`@Override`) следните методи:

| Метод | Предназначение |
|:--- |:--- |
| `toString()` | Връща текстово представяне на обекта (полезно за принтиране и дебъгване). |
| `equals(Object o)` | Определя кога два обекта се считат за логически еднакви. |
| `hashCode()` | Връща цифров идентификатор на обекта, необходим при работа с колекции (напр. `HashMap`). |

## Изброени типове (Enum)

**Enum** е специален вид клас, който се използва за дефиниране на фиксиран набор от константи (например дни от седмицата, цветове, статуси).

```java
enum Status {
    NEW, IN_PROGRESS, COMPLETED, CANCELLED
}
```

### Предимства на Enum:
* **Сигурност на типа (Type Safety):** Предотвратява задаването на невалидни стойности.
* **Четимост:** Прави кода по-лесен за разбиране от хора.
* **Вградени методи:** Притежава полезни методи като `values()` (връща масив от всички константи) и `valueOf()`.
