# Абстрактен клас

Абстрактният клас представлява специален клас, който служи като основа за създаване на други класове.

Той може да съдържа както реализирани методи, така и абстрактни методи – методи без реализация.

Абстрактният клас се декларира чрез ключовата дума abstract.

```
//абстрактен клас
abstract class Shape {

    abstract double area();

}

//Наследник на класа Shape
class Circle extends Shape {

    double radius;

    Circle(double radius) {
        this.radius = radius;
    }

    @Override
    double area() {
        return Math.PI * radius * radius;
    }
}
```
Използване:
```
Shape shape = new Circle(5);

System.out.println(shape.area());
```

### Правила

При работа с абстрактни класове трябва да се имат предвид следните особености:
* не могат да бъдат създавани обекти от абстрактен клас;
* абстрактният клас може да съдържа както абстрактни, така и конкретни методи;
* ако клас наследник не реализира всички абстрактни методи, той също трябва да бъде деклариран като абстрактен.

### Предимства

Използването на абстрактни класове:
* позволява извеждането на обща функционалност в родителски клас;
* намалява дублирането на код;
* гарантира, че наследниците ще реализират определени методи;
* улеснява използването на полиморфизъм.
