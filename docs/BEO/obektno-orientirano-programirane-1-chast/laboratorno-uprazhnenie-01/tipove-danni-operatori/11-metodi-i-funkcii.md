# Методи (функции)

При разработването на програми често се налага една и съща последователност от инструкции да бъде използвана многократно.

Вместо този код да се повтаря на различни места в програмата, той може да бъде отделен в метод (функция), който след това да бъде извикван при необходимост.

Използването на методи:
* намалява повторението на код;
* прави програмите по-четими;
* улеснява поддръжката и разширяването им.

В различните програмни езици често се използват термините *функция* и *метод*. В Java изпълнимите блокове код се наричат 
методи. В настоящата тема понятието се използва *единствено като начин за организиране на програмата*. Методите като 
поведение на обектите ще бъдат разгледани по-късно при изучаването на обектно-ориентираното програмиране.

Пример:
```java
public class Main {

    public static void main(String[] args) {

        printGreeting();

        printGreeting();

    }

    public static void printGreeting() {
        System.out.println("Hello!");
    }

}
```

Методът printGreeting() съдържа инструкциите за извеждане на текст на конзолата. Вместо тези инструкции да бъдат записвани многократно, те са 
отделени в самостоятелен метод, който може да бъде извикан неограничен брой пъти. В примера методът се извиква два 
пъти от метода main(), поради което текстът се извежда два пъти.

Методът не получава входни данни и не връща резултат. При всяко извикване изпълнява една и съща последователност от инструкции.

Друг пример за деклариране на функция:

```java
public static void printName(String name) {
    System.out.println("Hello, " + name);
}
```
Параметърът *name* позволява при различни извиквания към метода да се подава различна стойност.

Следващият пример демонстрира още една разновидност на функция:
```java
public static void printSum(int firstNumber, int secondNumber) {
    System.out.println(firstNumber + secondNumber);
}
```
 Последният пример илюстрира функция, която връща резултат:
 ```java
public static int calculateSum(int firstNumber, int secondNumber) {
    return firstNumber + secondNumber;
}
```

Използването на всички декларирани по-горе функции става по следния начин:
 ```java
public static void main(String[] args) {
    printGreeting();
    printName("Anna");
    printSum(5, 7);

    int result = calculateSum(5, 7);
    System.out.println(result);
}
```
