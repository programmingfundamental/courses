# Примитивни типове данни

**Типът данни** определя какви стойности може да съхранява една променлива, колко памет е необходима за нейното съхранение и какви операции могат да бъдат извършвани с нея.

Типовете данни се разделят на две основни групи: примитивни и референтни. В настоящото упражнение ще бъдат разгледани единствено примитивните типове данни. Референтните типове ще бъдат разгледани в следващите упражнения.

Следващата таблица обобщава съществуващите примитивни типове:

| Тип     | Размер | Стойности                       | Пример                   |
| ------- | ------ | ------------------------------- | ------------------------ |
| byte    | 8 bit  | -128 to 127                     | byte age = 25;           |
| short   | 16 bit | -32,768 to 32,767               | short depth = 10;        |
| int     | 32 bit | -2,147,483,648 to 2,147,483,647 | int count = 1200;        |
| long    | 64 bit | -2^63 to 2^63-1                 | long rgbColor = 123456L; |
| float   | 32 bit |                                 | float price = 10.25f;    |
| double  | 64 bit |                                 | double area = 33.14;     |
| char    | 16 bit | Unicode символ                  | char prefix = 's';       |
| boolean | -      | true/false                      | boolean isValid = true;  |

*Забележки:*

- *int* е най-често използваният целочислен тип;
- *double* е стандартният тип за реални числа;
- *char* съхранява един Unicode символ;
- *boolean* може да приема единствено стойности *true* и *false*.

## Променливи

Променливата представлява именувана област от паметта, в която се съхранява стойност от определен тип. В Java всяка променлива има тип, име и стойност.

Java е статично типизиран език. Типът на всяка променлива се определя по време на компилация и не може да бъде променян по време на изпълнение на програмата.

## Деклариране на променлива

Декларирането на променлива задава нейния тип и име:

```java
int number;
double price;
```

## Инициализация на променлива

Инициализацията представлява задаване на начална стойност на променливата:

```java
int number = 15;
double price = 13.78;
```

## Литерали

Литерал е фиксирана стойност, записана директнов програмния код. Тя се използва за инициализация на променливи, като аргумент на метод или като част от израз:

```java
10          // целочислен литерал
3.14        // литерал с плаваща запетая
'A'         // символен литерал
"Java"      // низов литерал
true        // булев литерал
null        // null литерал
```

## Присвояване на стойност

След като променливата е декларирана, на нея може да бъде присвоена нова стойност:

```java
int number;
number = 10;
number = 15;
```

## Дефиниция на променлива

В практиката често се използва изразът **дефиниране на променлива**. В контекста на Java това обикновено означава деклариране на променлива, а при наличие на начална стойност - деклариране с инициализация.

## final променливи

Ключовата дума *final* указва, че след първоначалното присвояване на стойност на променливата не може да бъде извършено ново присвояване:

```java
final int maxAttempts = 5;
```

Подобна декларация означава, че не е възможно на променливата *maxAttemts* след това да се присвои нова различна стойност.

## Константи

Константата представлява именувана стойност, която не се променя по време на изпълнение на програмата. В Java константите обикновено се реализират чрез *public static final* полета:

```java
public static final double PI = 3.141592653589793;
public static final int MAX_ATTEMPTS = 5;
```

## Var

Ключовата дума *var* позволява типът на локална променлива да бъде изведен автоматично от компилатора въз основа на присвоената стойност:

```java
var count = 10;    // int
var prefix = "T"   // char
var price = 1.33;  // double
```

Използването на *var* не означава, че Java става динамично типизиран език. Типът отново се определя по време на компилация и не може да бъде променян след това. Ако се използват дефинираните по-горе променливи, следният запис ще бъде невалиден:

```java
count = "text";
```

## Масиви

Масивът представлява структура от данни, предназначена за съхраняване на фиксиран брой елементи от един и същи тип. Всеки елемент се достъпва чрез индекс, като индексирането започва от 0.

## Дефиниране и инициализация на масив

Създаването на масив може да стане по следния начин:

```java
int[] numbers = {1, 5, 0, 10, 1};
```

Подобна дефиниция заделя място в паметта за променлива с име *numbers*, която представлява масив от целочислени стойности и има пет елемента с посочените съответни стойности.

## Достъп до елемент

Всеки елемент от даден масив се достъпва чрез неговия индекс:

```java
numbers[2] = 14;    // заменя нулевата стойност с нова
```

## Размер на масив

Всеки един масив има свойство *length*, което съдържа броя елементи в този масив:

```java
System.out.println(numbers.length);
```
