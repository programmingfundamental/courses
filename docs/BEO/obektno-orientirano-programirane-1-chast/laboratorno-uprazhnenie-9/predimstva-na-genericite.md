# Предимства на генериците

Използването на генерици предоставя няколко основни предимства:

- осигурява типова безопасност;
- открива част от грешките още по време на компилация;
- намалява необходимостта от явно преобразуване на типовете;
- позволява повторно използване на един и същ код с различни типове;
- улеснява създаването на общи структури от данни и алгоритми.

# Обобщение

| Понятие         		| Предназначение                               					|
|-----------------------|---------------------------------------------------------------|
| Генеричен клас        | Клас, който работи с параметризиран тип. 						|
| Генеричен метод       | Метод, който декларира собствен параметър на типа.            |
| Генеричен интерфейс   | Интерфейс, който работи с параметризиран тип.            		|
| Raw тип               | Генеричен тип, използван без параметър.                    	|
| Типова безопасност    | Проверка на коректността на типовете по време на компилация.  |
