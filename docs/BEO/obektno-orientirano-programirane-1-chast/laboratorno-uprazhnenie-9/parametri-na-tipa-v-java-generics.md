# Параметри на типа 

В Java съществуват утвърдени конвенции за именуване на параметрите на типа:

| Име   | Значение                  | Къде се ползва             |
|-------|---------------------------|-----------------------------|
| **T** | Type                      | общ тип                     |
| **E** | Element                   | колекции (List, Set и др.) |
| **K** | Key                       | ключове в Map               |
| **V** | Value                     | стойности в Map             |
| **R** | Result                    | резултати от функции        |
| **U** | Втори свободен тип        | когато T вече се използва   |      

---

Имената са конвенция, а не изискване на езика. Въпреки това е добра практика да се използват стандартните означения, тъй като правят кода по-разпознаваем.
