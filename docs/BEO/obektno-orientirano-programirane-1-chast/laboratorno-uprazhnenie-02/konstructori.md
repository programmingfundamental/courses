# Конструктори

Конструкторът е специален член на класа, който се изпълнява при създаването на нов обект. Неговото основно предназначение е да инициализира началното състояние на обекта.

Конструкторът се извиква автоматично при създаването на обект и не може да бъде извикан като обикновен метод.

## Основни характеристики

Конструкторът притежава следните характеристики:

- името му съвпада с името на класа;
- няма тип на връщаната стойност (дори void);
- може да приема параметри;
- един клас може да съдържа повече от един конструктор.

## Конструктор по подразбиране

Ако в класа не е дефиниран нито един конструктор, компилаторът автоматично създава конструктор по подразбиране, който не приема параметри.

Пример:

```java
class Student {

    // Полета
    String name;
    int facultyNumber;

}
```

В този случай компилаторът автоматично създава конструктор без параметри.

## Параметризиран конструктор

Конструкторът може да приема параметри, чрез които при създаването на обекта се задават началните стойности на неговите полета.

Пример:

```java
class Student {

    String name;
    int facultyNumber;

    Student(String studentName, int studentFacultyNumber) {
        name = studentName;
        facultyNumber = studentFacultyNumber;
    }
}
```

В примера стойностите на полетата се задават още при създаването на обекта.

## Предефиниране на конструктор по подразбиране

При дефиниране на собствен конструктор компилаторът не създава автоматично конструктор по подразбиране.

Ако е необходим конструктор без параметри, той трябва да бъде дефиниран изрично.

Пример:

```java
class Student {

    String name;
    int facultyNumber;

    Student() {}

    Student(String studentName, int studentFacultyNumber) {
        name = studentName;
        facultyNumber = studentFacultyNumber;
    }
}
```
