# Клас

## Декларация на клас

Класът се дефинира чрез ключовата дума class, следвана от неговото име и тяло, оградено с фигурни скоби.

```java
class Student {

}
```

Декларацията на един клас включва следните основни елементи:

- ключовата дума class, която указва началото на дефиницията;
- име на класа, избрано съгласно приетите конвенции за именуване;
- тяло на класа, оградено с фигурни скоби {}, в което се описват неговите членове.

Съгласно конвенциите в Java имената на класовете започват с главна буква и се записват в стил **PascalCase**.

В примера е деклариран клас с име Student. Към момента той не съдържа полета, конструктори или методи и представлява минимална декларация на клас.

## Структура на клас

Класът представлява съвкупност от данни и функционалност, които описват определен тип обекти. Неговата структура се състои от елементи, наречени членове на класа.

В най-общия случай един клас може да съдържа:

- полета (атрибути) – описват състоянието на обектите;
- конструктори – използват се за инициализиране на новосъздадените обекти;
- методи – реализират поведението на обектите.

Методите могат условно да бъдат разделени на:

- аксесори (getter методи) – използват се за извличане на стойността на дадено поле;
- мутатори (setter методи) – използват се за промяна на стойността на дадено поле;
- други методи, реализиращи поведението на обекта.

Пример:

```java
class Student {

    String name;

    String getName() {
        return name;
    }

    void setName(String name) {
        this.name = name;
    }
}
```

В зависимост от конкретната реализация един клас може да съдържа и други елементи, като вложени класове, изброени типове (enum), записи (record), статични блокове и др., които ще бъдат разгледани по-нататък.

Пример:

```java
class Student {

    // Полета
    String name;
    int facultyNumber;

    // Конструктор
    Student(String studentName, int studentFacultyNumber) {
        name = studentName;
        facultyNumber = studentFacultyNumber;
    }

    // Метод
    String fetchStudentInformation() {
         return name + ", " + facultyNumber;
    }
}
```

В примера класът Student съдържа две полета, един конструктор и един метод.

## Полета (атрибути)

Полетата представляват променливи, декларирани в тялото на класа. Те описват състоянието на обектите и съхраняват информацията, необходима за тяхното представяне.

Всяка инстанция на класа притежава собствен набор от стойности на своите полета.

## Конструктори

Конструкторът е специален член на класа, който се изпълнява при създаването на нов обект. Основното му предназначение е да инициализира полетата на обекта.

Подробно разглеждане на конструкторите ще бъде направено в следващата точка.

## Методи

Методите представляват именувани последователности от инструкции, които реализират поведението на обектите. Чрез тях се извършва обработка на данните, съхранявани в полетата на класа.

Методите могат да приемат параметри и при необходимост да връщат резултат.
